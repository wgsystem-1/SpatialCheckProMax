Cursor IDE에서 GitHub 원격 저장소 URL을 변경하는 방법
방법 1: 터미널에서 변경 (가장 간단)
# 현재 원격 저장소 확인git remote -v# 원격 URL 변경git remote set-url origin <새로운-GitHub-URL>
방법 2: Cursor/VS Code GUI에서 변경
Ctrl + Shift + P (명령 팔레트 열기)
Git: Remove Remote 입력 → origin 선택하여 삭제
Git: Add Remote 입력 → 새 URL 추가
방법 3: 설정 파일 직접 수정
.git/config 파일에서 [remote "origin"] 섹션의 url 값을 수정
